# 📞 Call Center Report - Excel Dashboard (2023)

## About This Project
An interactive Excel dashboard analyzing Call Center performance data for 2023 
across multiple cities — Cincinnati, Cleveland, and Columbus.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Line Chart, Bar Chart, Donut Chart)
- Slicers (interactive filtering)
- COUNTIFS, SUM formulas

## 📊 Key Metrics Tracked
- Total Calls: 1,000
- Total Amount: ₹96,623
- Total Revenue: ₹89,850
- Average Rating: 3.9
- Happy Callers: 307

## 📈 Key Insights
- **Saturday** had the highest call volume (40 calls)
- **R01 region** recorded the highest calls and amount
- **Female vs Male** caller comparison across 3 cities
- Monthly call trend shows peak in **January and October**
- Agent-wise performance tracked by call rank and amount rank
